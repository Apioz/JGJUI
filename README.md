# 智慧园区管理小程序

基于微信小程序的智慧园区管理平台，包含首页模块展示、协作中心、数据统计和个人中心等功能。

## 功能概览

### 首页
- **切换项目**：支持多项目/园区切换
- **轮播图**：展示园区宣传与通知
- **数据总览**：资产总数、待处理工单、今日能耗、食堂合格率
- **资产管理**：资产统计与快捷入口
- **物业管理**：报修工单、维保工单、巡检任务、我的工单
- **能源管理**：累计用水/用电数据
- **食堂管理**：三清三关、留样菜监测、晨检结果
- **消息通知**：顶部消息入口，未读红点提醒

### 协作
- 我的待办
- 我的工单
- 我的消息

### 数据
- 各模块汇总统计（资产、物业、能源、食堂）
- 趋势指标展示

### 我的
- 个人资料（所属项目园区、所属部门、工号等）
- 通讯录（支持搜索与拨号）

## 交互原型（浏览器预览）

无需微信开发者工具，直接在浏览器中打开可交互原型：

```
prototype/index.html
```

### 完整交互流程
1. **登录页** — 账号密码 / 手机号登录，忘记密码、注册账号
2. **选择项目** — 登录后选择权限内项目（延安东路300号等 4 个项目）
3. **首页** — 右上角切换项目，数据随项目变化；食堂/能源「查看更多」跳转数据总览
4. **协作** — 并列图标入口（待办、工单、消息）
5. **我的** — 编辑资料、修改密码、通讯录（支持增删改）

### 数据总览子页面
- **食堂管理** — 数据看板/每日菜谱、KPI、人员统计饼图、监测状态、营销折线图
- **能源管理** — 用电/用水切换、逐时趋势对比、用电类型统计

## 快速开始

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具，选择「导入项目」
3. 项目目录选择本仓库根目录
4. AppID 可使用测试号或替换 `project.config.json` 中的 `appid`
5. 点击「编译」即可预览

## 项目结构

```
├── app.js / app.json / app.wxss   # 小程序入口
├── pages/
│   ├── index/                     # 首页
│   ├── collaboration/             # 协作
│   ├── data/                      # 数据
│   ├── mine/                      # 我的
│   ├── todo/                      # 我的待办
│   ├── workorder/                 # 我的工单
│   ├── messages/                  # 我的消息
│   ├── notifications/             # 消息通知
│   ├── contacts/                  # 通讯录
│   └── profile/                   # 我的资料
├── utils/mock.js                  # 模拟数据
└── assets/icons/                  # TabBar 图标
```

## 后续扩展

当前使用 `utils/mock.js` 提供模拟数据，接入后端 API 时只需替换各页面的数据加载逻辑即可。
